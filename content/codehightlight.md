# **  **

### 1. 使用三个`显示代码块

> \```  
> public class helloWorld{  
>     public static void main(String[] args){  
>         System.out.println("Hello World!");  
>     }     
>}  
> \```

### 效果：

> ```  
> public class helloWorld{  
>     public static void main(String[] args){  
>         System.out.println("Hello World!");  
>     }     
>}  
> ```

### *REMARK：*
1. 代码块前后都需要三个\` ，并且三个`需要连在一起不能有空格

### 2. 使用上面方法，并在最后加上代码语言即可

> \```  java  
> public class helloWorld{  
>     public static void main(String[] args){  
>         System.out.println("Hello World!");  
>     }     
>}  
> \```

### 效果：

> ``` java 
> public class helloWorld{  
>     public static void main(String[] args){  
>         System.out.println("Hello World!");  
>     }     
>}  
> ```

### *REMARK：*
1. 语言名字需要放在第一个\```的后面
